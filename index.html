Internet Of Things

Related projects

Water Irrigation System

It is an IoT-based automate irrigation model. Using some IoT equipment we find the moisture level of the soil. And finally, It distributes the water throughout the lawn or terrace based on the soil condition. The Advanced is that while turning the motor on, Our model sent the user a message about motor status and soil moisture.

Smart Dustbin

Smart Dustbin is an IoT-based model. It works under Infrared rays. This means while a person or user stands in front of the Dustbin. It automatically opens and whenever the dustbin is loaded it sends the user a message that the dustbin has no space to load.

Computer Vision

Related projects

Mask Detection

It is a Computer Vision-based model to identify whether the person wears a mask or not. During the Covid pandemic situation, our govt. has advised us to wear the mask, And This model is used to detect whether the public wearing a mask or following covid avoidance regulation. I had made this mask detection model with the help of Yolo v5 Architecture.

Virtual Mouse Using Hands

Based on our Generation, our Technology also keeps updating like a wired mouse to wireless mouse and now we move to virtual reality. So we need to update our regular things with innovation. here I had made a virtual mouse which is a computer vision project. Using that we can control our computer's mouse virtually with normal hands just using a computer webcam.

Authentication Based Door Opener

This model is an IoT and Computer Vision-based model for home security. And it was mainly designed for home security and maintenance purposes. The Authentication based door opener authenticates whether the person is known or unknown. based on the authentication our model opens a door if the user is known or else it captures the victims face and sends it to the user.
